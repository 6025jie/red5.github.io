Red5 Media Server

Red5 is an open source media server written in Java that supports * Streaming Video (FLV, F4V, MP4, 3GP) * Streaming Audio (MP3, F4A, M4A, AAC) * Recording Client Streams (FLV and AVC+AAC in FLV container) * Shared Objects * Live Stream Publishing * Remoting * Protocols: RTMP, RTMPT, RTMPS, and RTMPE * WebSocket (ws and wss) * HLS * RTSP (From Axis-type cameras) [Red5 on StackOverflow](http://stackoverflow.com/questions/tagged/red5)
